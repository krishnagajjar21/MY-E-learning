The Core Web Technology

HTML (HyperText Markup Language) is the standard language for creating web pages.


1. Basic Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Welcome to HTML</h1>
  </body>
</html>

  • Every code you're ever visited is built with HTML code.

  • With this courses you'll be able to build your own web page.

  • HTML code is based on tags.


    • <button>

  • Tages use angle brackets <> Complete the button like this.

    • </button>

    We have to use <> for our tags.


  • The angle brackets <> surround the name of the element you want to add to the page. Elements like buttons,

    • text and images are added to web pages with different tags.

  • We have to use < at the start of our tag and close the tag with >.

  • For creating a valid button tag we use <button>.


    • Let's add the image togather.


  • You can use the image tag (<img>) to add images to a web page

    • Also for the image tag we can use <img> tag.

  • All we have to do is start with < and add the tag name, then close with >.


    • You need HTML tags to add different elements to a page.


    Like this:


    Button: <button>

    Image: <img>

    Paragraph: <p>

    Table: <table>
