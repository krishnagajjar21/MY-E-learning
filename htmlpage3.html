⭐ Headings



  • The text on web pages can be organized in different sections,with headings.

  • Headings also help search engines like Google know what your page is about to show it to

    the right people.

  • Headings in HTML come in different levels.

  • <h2> defines the most important heading.


    • let's code a level 1 heading


    
        ____ Heading 1 </h1>

  • First we have to use the <h1> starting tag and </h1> closing tag.

  • Now for display the text we can write anything we like between the <h1> and </h1> tags.

  • For our code we have to add <h1> tag to the blank space.


    • You can use up to 6 levels of heading in HTML. The tags for these heading element are <h1>,<h2>,<h3>,<h4>,<h5> and <h6>.
